*** AULA 3 ***
Rounded:
-> representa o border radius do objeto
-> rounded-full = maximo 

Border:
-> signfica o valor da border-width
-> t = top ; r = right ; l = left ; b = bottom
-> border-color-900 = representa o valor da cor

Ring:
-> sombra espaçada por volta do botão
-> ring-color-900
-> ring-2 = valor da width do ring

Bg:
-> Cor de fundo // background color
-> bg-cor-900
-> bg-opacity-100 = opacidade de 0 a 100
-> bg-gradient-{direction} = gradiente de fundo com direção de cor
